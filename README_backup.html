Train Track

Train Track is an android app which lists the train schedules for the Chennai MRTS.

Note : This app was created for personal use. I have not done extended testing.

Also this was hacked up in a weekend (under 32 hours to be specific).

Installation

  1. Download TrainTrack.apk
  2. Install it on your android phone
  3. Move the files in /time directory to your sdcard
  4. Open the App

Tested

  1. Nexus One
  2. HTC Legend

Note


A point to note, I have used flat files instead of a db file. The reason for that was :
  1. I was not going to update the time schedule.
  2. I didn’t need any security. (It’s for my personal use)
  3. Creating a db helper class was too much of a hassle.

Also, This schedule includes only one was transport from beach -> velachery and not vice versa. I would shortly add this as well.
